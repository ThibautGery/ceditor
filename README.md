# Ceditor

Editor in C written following this [tutorial](http://viewsourcecode.org/snaptoken/kilo/01.setup.html)


## Requirements

* [C compiler](http://viewsourcecode.org/snaptoken/kilo/01.setup.html#how-to-install-a-c-compiler)
* `make`

## Commands

### Compile

```sh
make
```

### Run

```sh
./kilo
```
